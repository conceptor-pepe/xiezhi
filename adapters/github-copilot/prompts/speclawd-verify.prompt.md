# Speclawd Verify

Verify whether the current change is ready for delivery.

Check:

- audit
- review
- test review
- error memory
- `REQ-*`, `DES-*`, `TASK-*`, `REV-*`, `CASE-*`, and `MEM-*` traceability
- product / architecture / senior engineer / test readiness conclusions
- white-box coverage conclusions
- required build, lint, test, and docs sync

Write results back into the change artifacts and state whether the change can move to commit summary.
